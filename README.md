# ued100-qantas-booking-prototype
UED100 Assessment 2 Qantas-inspired booking prototype

This project was created for UED100 Assessment 2.

It is a Qantas-inspired front-end web prototype developed using HTML, CSS, and JavaScript. The prototype focuses on flight booking, extra travel services, and booking support.
